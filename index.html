カコタムシステム開発室とは

活動目的

Kacotamの活動のシステム化を図り、活動効率の向上、ノウハウの蓄積を目指す。

活動内容

  • 団体WEBページの制作・運用
  • 基幹システムの制作・運用
  • プログラミング学習会の企画・運営（カコタムラボと協働）

使用言語・フレームワーク等

  • 団体WEBページ
    • Wordpress
    • Bootstrap
    • jQuery
    • Vue.js
  • 基幹システム
    • Django
    • Bootstrap
    • jQuery
    • PostgreSQL

活動要件

  • HTML・CSS・Javascript等でWEB制作をしたことのある方（高校生も可）
  • オンライン上でのコミュニケーション・作業が可能な方（Slack, Google Drive, Gitlab, Github...）

活動参加の流れ

  1. お問い合わせフォームから連絡する
  2. 団体について/開発室についての説明を受ける
  3. 各種ツールへのアカウント登録する
  4. 活動開始
